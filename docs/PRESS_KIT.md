# ReturnRider — Press kit (MKT-07)

## One-liner

**Flight tracker for your refunds** — connect Gmail once, never miss a return deadline.

## Elevator pitch (30 sec)

Americans leave billions on the table when return windows expire. ReturnRider reads shopping mail (read-only), surfaces deadlines, sends reminders, and helps you close the loop when refunds land. Privacy-first: shopping mail only, never sold.

## Founder story angle

Built after missing a return deadline on a routine online purchase — the refund was gone because no calendar reminder existed for “return by Sunday.” ReturnRider automates what people forget.
