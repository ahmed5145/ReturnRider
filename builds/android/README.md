# Android dev builds (local only — not committed)

Place EAS-downloaded APKs here for your own reference.

Example after download from [expo.dev builds](https://expo.dev/accounts/ahmed5145/projects/returnrider/builds):

```text
builds/android/returnrider-dev-2026-06-09.apk
```

Rename on copy, e.g.:

```cmd
move %USERPROFILE%\Downloads\application-*.apk builds\android\returnrider-dev.apk
```

Install on **Android emulator** (Android Studio → drag APK onto emulator) or share with a friend who has Android.

**iPhone only?** Use Expo Go for smoke tests; push/Plaid need dev client on Android or emulator.
